# Sporty
Sports and Fitness E-commerce web application.
---
# Backend

## 1. Technologies

* Java (Version 17)
* Spring Boot (Version 3.1.2)
  
## 2. Clone

```sh
git clone https://github.com/yasminenebti/Sporty.git
```

## 3. Build

```sh
cd backend
mvn clean package
```

## 4. Run

```sh
java -jar .\target\sporty.jar
```

# Frontend

## 1. Technologies
* React 18
* Javascript

## 2. Installation

```sh
cd frontend
npm install
```
## 2. Run

```bash
npm run dev
```
